Overview
========

The SDRAM module is designed for 16 bit read and write access of arbitrary length at up to 50MHz clock rates. 

Memory requirements
-------------------
+------------------+----------------------------------------+
| Resource         | Usage                            	    |
+==================+========================================+
| Stack            | 256 bytes                              |
+------------------+----------------------------------------+
| Program          | 10272 bytes                            |
+------------------+----------------------------------------+

Resource requirements
-------------------
+--------------+-------+
| Resource     | Usage |
+==============+=======+
| Channels     |   1   |
+--------------+-------+
| Timers       |   1   |
+--------------+-------+
| Clocks       |   1   |
+--------------+-------+
| Threads      |   1   |
+--------------+-------+
